# perlin-noise
Demo Program for Perlin noise

This program uses a customized OpenGL-Library made by Philipp Munz and Nicolas Hollmann

## how to build the program yourself
The program can be built using the Microsoft Visual Studio Build tools 2022 for amd64 architecture

If you want to build with VSCode, you need the C/C++ Extension pack, as well as the CMake Tools extension

If you want to build the Program with CLion, there is a CMake cache variable USING_CLION which you need to set to ON after loading the Project via the CMakeLists.txt
